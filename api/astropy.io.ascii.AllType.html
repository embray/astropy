AllType¶

class astropy.io.ascii.AllType[source] [edit on github]¶

Bases: astropy.io.ascii.StrType, astropy.io.ascii.FloatType, astropy.io.ascii.IntType

Subclass of all other data types.

This type is returned by convert_numpy if the given numpy type does not match StrType, FloatType, or IntType.
